# asw-825-spring-boot/f-restaurants-postgres

Questo progetto è una variante dell'applicazione **asw-825-spring-boot/d-restaurants**, 
in cui però viene utilizzata una base di dati [PostgreSQL](https://www.postgresql.org/). 


## Avvio della base di dati 

Nel nodo **dev** di un ambiente **developer** 

* posizionarsi nella cartella principale dell'applicazione 

* eseguire lo script `start-postgres.sh`

* per arrestare la base di dati, eseguire lo script `stop-postgres.sh` 
  (la base di dati viene cancellata)
