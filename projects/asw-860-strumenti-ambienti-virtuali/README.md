# Strumenti per la gestione di ambienti virtuali (asw-860) 

Questo "progetto" contiene la definizione di diversi *ambienti virtuali*, 
basati sull'uso di *strumenti per la gestione di ambienti virtuali* 
come **Vagrant**, insieme a **Oracle VM VirtualBox** oppure a **VMware Workstation**. 

* Strumenti per la gestione di ambienti virtuali, [versione per VirtualBox](virtualbox/)

* Strumenti per la gestione di ambienti virtuali, [versione per VMware Workstation](vmware/)
